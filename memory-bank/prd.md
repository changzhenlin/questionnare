# 问卷调查平台 — 产品需求文档（PRD）

**版本：** v1.0  
**日期：** 2026-04-22

---

## 1. 产品概述

一个面向个人用户的轻量在线问卷工具，支持快速创建问卷、分享链接回收答卷、查看统计结果。

---

## 2. 用户画像

| 维度 | 描述 |
|------|------|
| 身份 | 学生、自由职业者、小型活动组织者 |
| 核心诉求 | 快速发起问卷，收集反馈，查看结果 |
| 使用场景 | 课程调研、活动报名、意见投票、朋友聚会时间协调 |
| 痛点 | 现有工具广告多、操作复杂、需要注册多个平台 |
| 期望 | 打开即用、操作简单、数据直观 |

---

## 3. 核心功能

### 3.1 问卷创建
- 输入问卷标题和描述
- 添加题目：单选、多选、填空
- 拖拽调整题目顺序
- 保存问卷

### 3.2 问卷管理
- 我的问卷列表（创建时间倒序）
- 编辑已保存的问卷
- 删除问卷
- 问卷状态：草稿 / 已发布 / 已结束

### 3.3 发布与回收
- 发布问卷（状态变为"已发布"，生成访问链接）
- 复制分享链接
- 结束问卷（停止回收，状态变为"已结束"）
- 重新发布已结束的问卷

### 3.4 数据统计
- 总回收数、浏览数
- 各题目答卷分布（单选/多选：柱状图；填空：列表展示）
- 每份答卷详情查看

---

## 4. 功能优先级

| 优先级 | 功能 |
|--------|------|
| P0 | 创建/编辑/删除问卷、单选/多选/填空题型、发布/结束问卷、分享链接、基础统计 |
| P1 | 拖拽排序、答卷详情、重新发布 |
| P2 | 问卷描述富文本、二维码分享 |

---

## 5. 非功能需求

| 维度 | 要求 |
|------|------|
| 性能 | 页面加载 ≤ 2s；同时 100 人提交无卡顿 |
| 安全 | HTTPS 传输；防止 XSS/SQL 注入；匿名填写无需登录 |
| 兼容 | 支持 Chrome、Safari、微信内置浏览器 |
| 响应式 | 问卷填写页适配手机端 |

---

## 6. 成功指标

| 指标 | 目标 |
|------|------|
| 问卷创建到发布完成率 | ≥ 80% |
| 填写页加载时间 | ≤ 2s |
| 单问卷平均回收数 | ≥ 20 份 |

---

## 7. 潜在风险

| 风险 | 应对 |
|------|------|
| 垃圾答卷 | 单 IP 限答次数；设置答题密码 |
| 数据丢失 | 自动保存草稿；数据库定时备份 |
